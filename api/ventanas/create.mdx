## Overview

Creates a new ventana (betting window) in the system. Automatically creates an associated VENTANA user account. Part of the hierarchy: **Banca → Ventana → Vendedor**.

## Authorization

<Info>**Required role:** ADMIN</Info>

## Request Body

<ParamField body="bancaId" type="string" required>
  UUID of the parent banca
</ParamField>

<ParamField body="name" type="string" required>
  Ventana name (2-100 characters)
</ParamField>

<ParamField body="code" type="string">
  Unique ventana code (2-10 characters, optional but recommended)
</ParamField>

<ParamField body="username" type="string" required>
  Username for the VENTANA user account (3-100 characters, alphanumeric, underscores, and hyphens only)
</ParamField>

<ParamField body="password" type="string" required>
  Password for the VENTANA user account (minimum 6 characters)
</ParamField>

<ParamField body="commissionMarginX" type="number">
  Commission margin multiplier (must be >= 0)
</ParamField>

<ParamField body="address" type="string">
  Physical address (max 255 characters)
</ParamField>

<ParamField body="phone" type="string">
  Contact phone number (max 20 characters)
</ParamField>

<ParamField body="email" type="string">
  Contact email address (must be valid email format)
</ParamField>

<ParamField body="isActive" type="boolean" default="true">
  Whether the ventana is active
</ParamField>

<ParamField body="settings" type="object">
  Ventana-specific settings
  
  <Expandable title="Settings object">
    <ParamField body="settings.print" type="object">
      Print configuration settings
      
      <Expandable title="Print settings">
        <ParamField body="settings.print.name" type="string">
          Printer name (max 100 characters)
        </ParamField>
        
        <ParamField body="settings.print.phone" type="string">
          Phone to print on receipts (max 32 characters)
        </ParamField>
        
        <ParamField body="settings.print.width" type="number">
          Paper width: 58 or 88 (mm)
        </ParamField>
        
        <ParamField body="settings.print.footer" type="string">
          Footer text (max 200 characters)
        </ParamField>
        
        <ParamField body="settings.print.barcode" type="boolean">
          Whether to print barcodes
        </ParamField>
        
        <ParamField body="settings.print.bluetoothMacAddress" type="string">
          Bluetooth MAC address (format: AA:BB:CC:DD:EE:FF or AABBCCDD)
        </ParamField>
      </Expandable>
    </ParamField>
    
    <ParamField body="settings.theme" type="string">
      UI theme: "light" or "dark"
    </ParamField>
  </Expandable>
</ParamField>

## Notes

- Automatically creates a VENTANA role user with the provided username/password
- Code must be unique across all ventanas
- Parent banca must exist and be active
- Username validation: alphanumeric, underscores, and hyphens only (see src/api/v1/validators/ventana.validator.ts:89-94)
- Activity is logged for audit purposes (see src/api/v1/controllers/ventana.controller.ts:6-15)
